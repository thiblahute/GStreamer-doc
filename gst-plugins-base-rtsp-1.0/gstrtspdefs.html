GST_RTSP_CHECK 

#define GST_RTSP_CHECK(stmt, label)  \
G_STMT_START { \
  if (G_UNLIKELY ((res = (stmt)) != GST_RTSP_OK)) \
    goto label; \
} G_STMT_END

Macro that checks the return value of stmt and jumps to label when it does not equal GST_RTSP_OK.

Parameters:

stmt –

a statement
