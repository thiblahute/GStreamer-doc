function Gst.Rtsp.Global.RtspRangeGetTimes(range: Gst.Rtsp.RTSPTimeRange, min: Gst.ClockTime, max: Gst.ClockTime): {
    // javascript wrapper for 'gst_rtsp_range_get_times'
}

Retrieve the minimum and maximum values from range converted to Gst.ClockTime in min and max.

A value of Gst.Constants.CLOCK_TIME_NONE will be used to signal Gst.Rtsp.RTSPTimeType.Now and Gst.Rtsp.RTSPTimeType.End for min and max respectively.

UTC times will be converted to nanoseconds since 1900.

Parameters:

range ( Gst.Rtsp.RTSPTimeRange ) –

a Gst.Rtsp.RTSPTimeRange

min ( Gst.ClockTime ) –

result minimum Gst.ClockTime

max ( Gst.ClockTime ) –

result maximum Gst.ClockTime

Returns ( bool ) –

true on success.

Since : 1.2
