query-position 

guint64
query_position_callback (NleComposition * comp,
                         gpointer udata)

A signal that has to be connected and which should return the current position of the pipeline.

This signal is used in order to know the current position of the whole pipeline so it is user's responsability to give that answer as there is no other way to precisely know the position in the whole pipeline.
