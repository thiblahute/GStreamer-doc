Properties

uri 

“uri” str

The uri of the file/resource to use. You can set a start index, a stop index and a sequence pattern. The format is &lt;multifile://start:stop@location-pattern&gt;. The pattern uses printf string formating.

Example uris:

multifile:///home/you/image%03d.jpg

multifile://20:50@/home/you/sequence/%04d.png

Flags : Read / Write / Construct Only
