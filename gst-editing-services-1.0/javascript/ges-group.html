GESGroup

A GES.Group is an object which controls one or more GES.Clip in one or more GES.Layer(s).

To instanciate a group, you should use the ges_container_group method, this will be responsible for deciding what subclass of GES.Container should be instaciated to group the various GES.TimelineElement passed in parametter.
