Force processing (disables passthrough)

Flags : Read / Write

Default value : false

use-sink-caps 

“use-sink-caps” gboolean

Use the sink caps for defining the output format

Flags : Read / Write

Default value : false

width 

“width” gint

Width of frames in raw stream

Flags : Read / Write

Default value : 320

height 

“height” gint

Height of frames in raw stream

Flags : Read / Write

Default value : 240

format 

“format” GstVideoFormat

Format of frames in raw stream

Flags : Read / Write

Default value : i420 (2)

pixel-aspect-ratio 

“pixel-aspect-ratio” GstFraction

Pixel aspect ratio of frames in raw stream

Flags : Read / Write

Default value : 1/1

framerate 

“framerate” GstFraction

Rate of frames in raw stream

Flags : Read / Write

Default value : 25/1

interlaced 

“interlaced” gboolean

True if frames in raw stream are interlaced

Flags : Read / Write

Default value : false

top-field-first 

“top-field-first” gboolean

True if top field in frames in raw stream come first (not used if frames aren't interlaced)

Flags : Read / Write

Default value : false

plane-strides 

“plane-strides” GstValueArray

Strides of the planes in bytes (e.g. plane-strides="<320,320>")

Flags : Read / Write

plane-offsets 

“plane-offsets” GstValueArray

Offsets of the planes in bytes (e.g. plane-offset="<0,76800>")

Flags : Read / Write

frame-size 

“frame-size” guint

Size of a frame (0 = frames are tightly packed together)

Flags : Read / Write

Default value : 0
