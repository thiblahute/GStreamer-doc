GstDataQueueClass

GstBase.DataQueue is an object that handles threadsafe queueing of objects. It also provides size-related functionality. This object should be used for any Gst.Element that wishes to provide some sort of queueing functionality.

gst_data_queue_push (not introspectable) 

def gst_data_queue_push (not introspectable) (self, item):
    #python wrapper for 'gst_data_queue_push'

Pushes a GstDataQueueItem (not introspectable) (or a structure that begins with the same fields) on the queue. If the queue is full, the call will block until space is available, OR the queue is set to flushing state. MT safe.

Note that this function has slightly different semantics than Gst.Pad.push and Gst.Pad.push_event: this function only takes ownership of item and the Gst.MiniObject contained in item if the push was successful. If False is returned, the caller is responsible for freeing item and its contents.

gst_data_queue_set_flushing (not introspectable) 

def gst_data_queue_set_flushing (not introspectable) (self, flushing):
    #python wrapper for 'gst_data_queue_set_flushing'

Sets the queue to flushing state if flushing is True. If set to flushing state, any incoming data on the queue will be discarded. Any call currently blocking on gst_data_queue_push (not introspectable) or gst_data_queue_pop (not introspectable) will return straight away with a return value of False. While the queue is in flushing state, all calls to those two functions will return False.

MT Safe.
