size of data

identification data at start of stream

}

Convenience function using Gst.Tag.TagFromVorbisTag, parsing a vorbis comment string into the right type and adding it to the given taglist list.

Unknown vorbiscomment tags will be added to the tag list in form of a Gst.Constants.TAG_EXTENDED_COMMENT.

Parameters:

list ( Gst.TagList ) –

a Gst.TagList

tag ( string ) –

a vorbiscomment tag string (key in key=value), must be valid UTF-8

value ( string ) –

a vorbiscomment value string (value in key=value), must be valid UTF-8
