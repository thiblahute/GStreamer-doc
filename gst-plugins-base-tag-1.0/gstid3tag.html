}

Adds an image from an ID3 APIC frame (or similar, such as used in FLAC) to the given tag list. Also see Gst.Tag.TagImageDataToImageSample for more information on image tags in GStreamer.

Parameters:

tag_list ( Gst.TagList ) –

a tag list

image_data ( byte ) –

the (encoded) image

image_data_len ( uint ) –

the length of the encoded image data at image_data

id3_picture_type ( uint ) –

picture type as per the ID3 (v2.4.0) specification for the APIC frame (0 = unknown/other)

Returns ( bool ) –

true if the image was processed, otherwise false
