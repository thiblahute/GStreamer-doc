GstVideoChromaResample

The functions GstVideo.video_chroma_from_string and GstVideo.video_chroma_to_string convert between GstVideo.VideoChromaSite and string descriptions.

GstVideo.VideoChromaResample is a utility object for resampling chroma planes and converting between different chroma sampling sitings.
