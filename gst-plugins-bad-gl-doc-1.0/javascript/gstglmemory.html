GstGlMemory

GstGLMemory is a GstGL.GLBaseMemory subclass providing support for the mapping of OpenGL textures.

GstGL.GLMemory is created or wrapped through GstGL.prototype.gl_base_memory_alloc with GstGL.GLVideoAllocationParams.

Data is uploaded or downloaded from the GPU as is necessary.
