GstGLRenderBuffer

GstGLRenderbuffer is a GstGL.GLBaseMemory subclass providing support for OpenGL renderbuffers.

GstGLRenderbuffer is created or wrapped through GstGL.prototype.gl_base_memory_alloc with GstGLRenderbufferAllocationParams.
