Properties

blocksize 

“blocksize” guint

Size in bytes to read per buffer (-1 = default)

Flags : Read / Write

Default value : 4096

buffer-size 

“buffer-size” guint

Size of internal buffer in number of video frames

Flags : Read / Write

Default value : 5

connection 

“connection” GstDecklinkConnection

Video input connection to use

Flags : Read / Write / Construct

Default value : auto (0)

device-number 

“device-number” gint

Output device instance to use

Flags : Read / Write / Construct

Default value : 0

do-timestamp 

“do-timestamp” gboolean

Apply current stream time to buffers

Flags : Read / Write

Default value : false

drop-no-signal-frames 

“drop-no-signal-frames” gboolean

Drop frames that are marked as having no input signal

Flags : Read / Write

Default value : false

mode 

“mode” GstDecklinkModes

Video Mode to use for playback

Flags : Read / Write / Construct

Default value : auto (0)

name 

“name” gchar *

The name of the object

Flags : Read / Write / Construct

Default value : decklinkvideosrc0

num-buffers 

“num-buffers” gint

Number of buffers to output before sending EOS (-1 = unlimited)

Flags : Read / Write

Default value : -1

output-stream-time 

“output-stream-time” gboolean

Output stream time directly instead of translating to pipeline clock

Flags : Read / Write

Default value : false

parent 

“parent” GstObject *

The parent of the object

Flags : Read / Write

signal 

“signal” gboolean

True if there is a valid input signal available

Flags : Read

Default value : true

skip-first-time 

“skip-first-time” guint64

Skip that much time of initial frames after starting

Flags : Read / Write
