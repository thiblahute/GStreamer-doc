GESAudioTrack

Sane default properties to specify and fixate the output stream are set as restriction-caps. It is advised, to modify these properties, to use GES.Track.prototype.update_restriction_caps, setting them directly is possible through GES.Track.prototype.set_restriction_caps, but not specifying one of them can lead to negotiation issues, only use that function if you actually know what you're doing :)

The default properties are:

  • format: S32LE
  • channels: 2
  • rate: 44100
  • layout: interleaved
