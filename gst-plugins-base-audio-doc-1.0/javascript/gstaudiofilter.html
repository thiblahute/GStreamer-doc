GstAudioFilter

GstAudio.AudioFilter is a GstBaseTransform (not introspectable)-derived base class for simple audio filters, ie. those that output the same format that they get as input.

GstAudio.AudioFilter will parse the input format for you (with error checking) before calling your setup function. Also, elements deriving from GstAudio.AudioFilter may use GstAudio.AudioFilterClass.prototype.add_pad_templates from their class_init function to easily configure the set of caps/formats that the element is able to handle.

Derived classes should override the GstAudioFilterClass.setup (not introspectable)() and GstBaseTransformClass.transform_ip and/or GstBaseTransformClass.transform (not introspectable)() virtual functions in their class_init function.
