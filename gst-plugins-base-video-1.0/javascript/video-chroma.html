GstVideoChromaResample

The functions GstVideo.prototype.video_chroma_from_string and GstVideo.prototype.video_chroma_to_string convert between GstVideo.VideoChromaSite and string descriptions.

GstVideoChromaResample is a utility object for resampling chroma planes and converting between different chroma sampling sitings.
