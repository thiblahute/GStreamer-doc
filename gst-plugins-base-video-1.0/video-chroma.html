GstVideoChromaResample

The functions Gst.Video.Global.VideoChromaFromString and Gst.Video.Global.VideoChromaToString convert between Gst.Video.VideoChromaSite and string descriptions.

Gst.Video.VideoChromaResample is a utility object for resampling chroma planes and converting between different chroma sampling sitings.

gst_video_chroma_resample_new (not introspectable) 

Gst.Video.VideoChromaResample *
gst_video_chroma_resample_new (not introspectable) (Gst.Video.VideoChromaMethod method,
                                                    Gst.Video.VideoChromaSite site,
                                                    Gst.Video.VideoChromaFlags flags,
                                                    Gst.Video.VideoFormat format,
                                                    int h_factor,
                                                    int v_factor)

Create a new resampler object for the given parameters. When h_factor or v_factor is > 0, upsampling will be used, otherwise subsampling is performed.

Parameters:

method –

a Gst.Video.VideoChromaMethod

site –

a Gst.Video.VideoChromaSite

flags –

Gst.Video.VideoChromaFlags

format –

the Gst.Video.VideoFormat

h_factor –

horizontal resampling factor

v_factor –

vertical resampling factor

Returns –

a new Gst.Video.VideoChromaResample that should be freed with Gst.Video.VideoChromaResample.Free after usage.
