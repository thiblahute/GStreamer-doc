GstVideoInfo

Gst.Video.VideoInfo

Information describing image properties. This information can be filled in from GstCaps with Gst.Video.VideoInfo.FromCaps. The information is also used to store the specific video info when mapping a video frame with Gst.Video.VideoFrame.Map.

Use the provided macros to access the info in this structure.

Gst.Video.VideoInfo.SetFormat 

function Gst.Video.VideoInfo.SetFormat(format: Gst.Video.VideoFormat, width: uint, height: uint): {
    // javascript wrapper for 'gst_video_info_set_format'
}

Set the default info for a video frame of format and width and height.

Note: This initializes info first, no values are preserved. This function does not set the offsets correctly for interlaced vertically subsampled formats.

Parameters:

info ( Gst.Video.VideoInfo ) –

a Gst.Video.VideoInfo

format ( Gst.Video.VideoFormat ) –

the format

width ( uint ) –

a width

height ( uint ) –

a height

Returns ( bool ) –

false if the returned video info is invalid, e.g. because the size of a frame can't be represented as a 32 bit integer (Since: 1.12)
