GstRTSPStream

The GstRtspServer.RTSPStream object manages the data transport for one stream. It is created from a payloader element and a source pad that produce the RTP packets for the stream.

With GstRtspServer.RTSPStream.join_bin the streaming elements are added to the bin and rtpbin. GstRtspServer.RTSPStream.leave_bin removes the elements again.

The GstRtspServer.RTSPStream will use the configured addresspool, as set with GstRtspServer.RTSPStream.set_address_pool, to allocate multicast addresses for the stream. With GstRtspServer.RTSPStream.get_multicast_address you can get the configured address.

With gst_rtsp_stream_get_server_port () you can get the port that the server will use to receive RTCP. This is the part that the clients will use to send RTCP to.

With GstRtspServer.RTSPStream.add_transport destinations can be added where the stream should be sent to. Use GstRtspServer.RTSPStream.remove_transport to remove the destination again.

GstRtspServer.RTSPStream.transport_filter 

def GstRtspServer.RTSPStream.transport_filter (self, func, user_data):
    #python wrapper for 'gst_rtsp_stream_transport_filter'

Call func for each transport managed by stream. The result value of func determines what happens to the transport. func will be called with stream locked so no further actions on stream can be performed from func.

If func returns GstRtspServer.RTSPFilterResult, the transport will be removed from stream.

If func returns GstRtspServer.RTSPFilterResult, the transport will remain in stream.

If func returns GstRtspServer.RTSPFilterResult, the transport will remain in stream but will also be added with an additional ref to the result GLib.List of this function..

When func is None, GstRtspServer.RTSPFilterResult will be assumed for each transport.

Parameters:

stream ( GstRtspServer.RTSPStream ) –

a GstRtspServer.RTSPStream

func ( GstRtspServer.RTSPStreamTransportFilterFunc ) –

a callback

user_data ( object ) –

user data passed to func

Returns ( [GstRtspServer.RTSPStreamTransport ] ) –

a GLib.List with all transports for which func returned GstRtspServer.RTSPFilterResult. After usage, each element in the GLib.List should be unreffed before the list is freed.

GstRtspServer.RTSPFilterResult
(*GstRtspServer.RTSPStreamTransportFilterFunc) (GstRtspServer.RTSPStream,
                                                GstRtspServer.RTSPStreamTransport,
                                                object)

This function will be called by the GstRtspServer.RTSPStream.transport_filter. An implementation should return a value of GstRtspServer.RTSPFilterResult.

When this function returns GstRtspServer.RTSPFilterResult, trans will be removed from stream.

A return value of GstRtspServer.RTSPFilterResult will leave trans untouched in stream.

A value of GstRtspServer.RTSPFilterResult will add trans to the result GLib.List of GstRtspServer.RTSPStream.transport_filter.
