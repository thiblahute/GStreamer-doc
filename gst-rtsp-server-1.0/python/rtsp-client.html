GstRTSPClient

The client object handles the connection with a client for as long as a TCP connection is open.

A GstRtspServer.RTSPClient is created by GstRtspServer.RTSPServer when a new connection is accepted and it inherits the GstRtspServer.RTSPMountPoints, GstRtspServer.RTSPSessionPool, GstRtspServer.RTSPAuth and GstRtspServer.RTSPThreadPool from the server.

The client connection should be configured with the GstRtsp.RTSPConnection using GstRtspServer.RTSPClient.set_connection before it can be attached to a GLib.MainContext using GstRtspServer.RTSPClient.attach. From then on the client will handle requests on the connection.

Use GstRtspServer.RTSPClient.session_filter to iterate or modify all the GstRtspServer.RTSPSession objects managed by the client object.

GstRtspServer.RTSPClient.session_filter 

def GstRtspServer.RTSPClient.session_filter (self, func, user_data):
    #python wrapper for 'gst_rtsp_client_session_filter'

Call func for each session managed by client. The result value of func determines what happens to the session. func will be called with client locked so no further actions on client can be performed from func.

If func returns GstRtspServer.RTSPFilterResult, the session will be removed from client.

If func returns GstRtspServer.RTSPFilterResult, the session will remain in client.

If func returns GstRtspServer.RTSPFilterResult, the session will remain in client but will also be added with an additional ref to the result GLib.List of this function..

When func is None, GstRtspServer.RTSPFilterResult will be assumed for each session.

Parameters:

client:

a GstRtspServer.RTSPClient

a callback

user data passed to func

object
Returns:

GstRtspServer.RTSPClientSessionFilterFunc 

GstRtspServer.RTSPFilterResult
(*GstRtspServer.RTSPClientSessionFilterFunc) (GstRtspServer.RTSPClient,
                                              GstRtspServer.RTSPSession,
                                              object)

This function will be called by the GstRtspServer.RTSPClient.session_filter. An implementation should return a value of GstRtspServer.RTSPFilterResult.

When this function returns GstRtspServer.RTSPFilterResult, sess will be removed from client.

A return value of GstRtspServer.RTSPFilterResult will leave sess untouched in client.

A value of GstRtspServer.RTSPFilterResult will add sess to the result GLib.List of GstRtspServer.RTSPClient.session_filter.
