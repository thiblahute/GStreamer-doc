GstRTSPSessionPool

The GstRtspServer.RTSPSessionPool object manages a list of GstRtspServer.RTSPSession objects.

The maximum number of sessions can be configured with GstRtspServer.RTSPSessionPool.set_max_sessions. The current number of sessions can be retrieved with GstRtspServer.RTSPSessionPool.get_n_sessions.

Use GstRtspServer.RTSPSessionPool.create to create a new GstRtspServer.RTSPSession object. The session object can be found again with its id and GstRtspServer.RTSPSessionPool.find.

All sessions can be iterated with GstRtspServer.RTSPSessionPool.filter.

Run GstRtspServer.RTSPSessionPool.cleanup periodically to remove timed out sessions or use GstRtspServer.RTSPSessionPool.create_watch to be notified when session cleanup should be performed.

Last reviewed on 2013-07-11 (1.0.0)

GstRtspServer.RTSPSessionPool.filter 

def GstRtspServer.RTSPSessionPool.filter (self, func, user_data):
    #python wrapper for 'gst_rtsp_session_pool_filter'

Call func for each session in pool. The result value of func determines what happens to the session. func will be called with the session pool locked so no further actions on pool can be performed from func.

If func returns GstRtspServer.RTSPFilterResult, the session will be set to the expired state with gst_rtsp_session_set_expired and removed from pool.

If func returns GstRtspServer.RTSPFilterResult, the session will remain in pool.

If func returns GstRtspServer.RTSPFilterResult, the session will remain in pool but will also be added with an additional ref to the result GList of this function..

When func is None, GstRtspServer.RTSPFilterResult will be assumed for all sessions.

Parameters:

pool ( GstRtspServer.RTSPSessionPool ) –

a GstRtspServer.RTSPSessionPool

func ( GstRtspServer.RTSPSessionPoolFilterFunc ) –

a callback

user_data ( object ) –

user data passed to func

Returns ( GLib.List ) –

a GList with all sessions for which func returned GstRtspServer.RTSPFilterResult. After usage, each element in the GList should be unreffed before the list is freed.

GstRtspServer.RTSPSessionPoolFilterFunc 

GstRtspServer.RTSPFilterResult
(*GstRtspServer.RTSPSessionPoolFilterFunc) (GstRtspServer.RTSPSessionPool,
                                            GstRtspServer.RTSPSession,
                                            object)

This function will be called by the GstRtspServer.RTSPSessionPool.filter. An implementation should return a value of GstRtspServer.RTSPFilterResult.

When this function returns GstRtspServer.RTSPFilterResult, session will be removed from pool.

A return value of GstRtspServer.RTSPFilterResult will leave session untouched in pool.

A value of GST_RTSP_FILTER_REF will add session to the result GLib.List of GstRtspServer.RTSPSessionPool.filter.

a GstRtspServer.RTSPSessionPool object
