GstRTSPMountPoints

A GstRtspServer.RTSPMountPoints object maintains a relation between paths and GstRtspServer.RTSPMediaFactory objects. This object is usually given to GstRtspServer.RTSPClient and used to find the media attached to a path.

With gst_rtsp_mount_points_add_factory () and GstRtspServer.RTSPMountPoints.remove_factory, factories can be added and removed.

With GstRtspServer.RTSPMountPoints.match you can find the GstRtspServer.RTSPMediaFactory object that completely matches the given path.

Last reviewed on 2013-07-11 (1.0.0)

def GstRtspServer.RTSPMountPoints.add_factory (self, path, factory):
    #python wrapper for 'gst_rtsp_mount_points_add_factory'

Attach factory to the mount point path in mounts.

path is of the form (/node)+. Any previous mount point will be freed.

Ownership is taken of the reference on factory so that factory should not be used after calling this function.
