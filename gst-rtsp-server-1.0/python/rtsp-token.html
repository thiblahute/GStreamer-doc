GstRTSPToken

A GstRtspServer.RTSPToken contains the permissions and roles of the user performing the current request. A token is usually created when a user is authenticated by the GstRtspServer.RTSPAuth object and is then placed as the current token for the current request.

GstRtspServer.RTSPAuth can use the token and its contents to check authorization for various operations by comparing the token to the GstRtspServer.RTSPPermissions of the object.

The accepted values of the token are entirely defined by the GstRtspServer.RTSPAuth object that implements the security policy.
