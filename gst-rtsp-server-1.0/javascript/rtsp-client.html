GstRTSPClient

The client object handles the connection with a client for as long as a TCP connection is open.

A GstRtspServer.RTSPClient is created by GstRtspServer.RTSPServer when a new connection is accepted and it inherits the GstRtspServer.RTSPMountPoints, GstRtspServer.RTSPSessionPool, GstRtspServer.RTSPAuth and GstRtspServer.RTSPThreadPool from the server.

The client connection should be configured with the GstRTSPConnection using GstRtspServer.RTSPClient.prototype.set_connection before it can be attached to a GMainContext using GstRtspServer.RTSPClient.prototype.attach. From then on the client will handle requests on the connection.

Use GstRtspServer.RTSPClient.prototype.session_filter to iterate or modify all the GstRtspServer.RTSPSession objects managed by the client object.

Last reviewed on 2013-07-11 (1.0.0)

GstRtspServer.RTSPClient.prototype.session_filter 

function GstRtspServer.RTSPClient.prototype.session_filter(func: GstRtspServer.RTSPClientSessionFilterFunc, user_data: Object): {
    // javascript wrapper for 'gst_rtsp_client_session_filter'
}

Call func for each session managed by client. The result value of func determines what happens to the session. func will be called with client locked so no further actions on client can be performed from func.

If func returns GstRtspServer.RTSPFilterResult, the session will be removed from client.

If func returns GstRtspServer.RTSPFilterResult, the session will remain in client.

If func returns GstRtspServer.RTSPFilterResult, the session will remain in client but will also be added with an additional ref to the result GList of this function..

When func is null, GstRtspServer.RTSPFilterResult will be assumed for each session.

Parameters

client:

a GstRtspServer.RTSPClient

GstRtspServer.RTSPClient *

func:

a callback

GstRtspServer.RTSPClientSessionFilterFunc

user_data:

user data passed to func

Object

Returns:

a GList with all sessions for which func returned GstRtspServer.RTSPFilterResult. After usage, each element in the GList should be unreffed before the list is freed.

GstRtspServer.RTSPFilterResult
(*GstRtspServer.RTSPClientSessionFilterFunc) (GstRtspServer.RTSPClient *,
                                              GstRtspServer.RTSPSession *,
                                              Object)

This function will be called by the GstRtspServer.RTSPClient.prototype.session_filter. An implementation should return a value of GstRtspServer.RTSPFilterResult.

When this function returns GstRtspServer.RTSPFilterResult, sess will be removed from client.

A return value of GstRtspServer.RTSPFilterResult will leave sess untouched in client.

A value of GstRtspServer.RTSPFilterResult will add sess to the result GList of GstRtspServer.RTSPClient.prototype.session_filter.
