GstRTSPSessionPool

The GstRtspServer.RTSPSessionPool object manages a list of GstRtspServer.RTSPSession objects.

The maximum number of sessions can be configured with GstRtspServer.RTSPSessionPool.prototype.set_max_sessions. The current number of sessions can be retrieved with GstRtspServer.RTSPSessionPool.prototype.get_n_sessions.

Use GstRtspServer.RTSPSessionPool.prototype.create to create a new GstRtspServer.RTSPSession object. The session object can be found again with its id and GstRtspServer.RTSPSessionPool.prototype.find.

All sessions can be iterated with GstRtspServer.RTSPSessionPool.prototype.filter.

Run GstRtspServer.RTSPSessionPool.prototype.cleanup periodically to remove timed out sessions or use GstRtspServer.RTSPSessionPool.prototype.create_watch to be notified when session cleanup should be performed.

Last reviewed on 2013-07-11 (1.0.0)

GstRtspServer.RTSPSessionPool.prototype.filter 

function GstRtspServer.RTSPSessionPool.prototype.filter(func: GstRtspServer.RTSPSessionPoolFilterFunc, user_data: Object): {
    // javascript wrapper for 'gst_rtsp_session_pool_filter'
}

Call func for each session in pool. The result value of func determines what happens to the session. func will be called with the session pool locked so no further actions on pool can be performed from func.

If func returns GstRtspServer.RTSPFilterResult, the session will be set to the expired state with gst_rtsp_session_set_expired and removed from pool.

If func returns GstRtspServer.RTSPFilterResult, the session will remain in pool.

If func returns GstRtspServer.RTSPFilterResult, the session will remain in pool but will also be added with an additional ref to the result GList of this function..

When func is null, GstRtspServer.RTSPFilterResult will be assumed for all sessions.

GstRtspServer.RTSPSessionPoolFilterFunc 

GstRtspServer.RTSPFilterResult
(*GstRtspServer.RTSPSessionPoolFilterFunc) (GstRtspServer.RTSPSessionPool,
                                            GstRtspServer.RTSPSession,
                                            Object)

This function will be called by the GstRtspServer.RTSPSessionPool.prototype.filter. An implementation should return a value of GstRtspServer.RTSPFilterResult.

When this function returns GstRtspServer.RTSPFilterResult, session will be removed from pool.

A return value of GstRtspServer.RTSPFilterResult will leave session untouched in pool.

A value of GST_RTSP_FILTER_REF will add session to the result GList of GstRtspServer.RTSPSessionPool.prototype.filter.
