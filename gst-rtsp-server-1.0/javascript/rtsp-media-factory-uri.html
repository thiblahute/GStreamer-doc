GstRTSPMediaFactoryURI

This specialized GstRtspServer.RTSPMediaFactory constructs media pipelines from a URI, given with GstRtspServer.RTSPMediaFactoryURI.prototype.set_uri.

It will automatically demux and payload the different streams found in the media at URL.

Last reviewed on 2013-07-11 (1.0.0)
