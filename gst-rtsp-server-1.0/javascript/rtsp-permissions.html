GstRTSPPermissions

The GstRtspServer.RTSPPermissions object contains an array of roles and associated permissions. The roles are represented with a string and the permissions with a generic Gst.Structure.

The permissions are deliberately kept generic. The possible values of the roles and Gst.Structure keys and values are only determined by the GstRtspServer.RTSPAuth object that performs the checks on the permissions and the current GstRtspServer.RTSPToken.

As a convenience function, GstRtspServer.RTSPPermissions.prototype.is_allowed can be used to check if the permissions contains a role that contains the boolean value TRUE for the the given key.
