GstRTSPSessionMedia

The GstRtspServer.RTSPSessionMedia object manages a GstRtspServer.RTSPMedia with a given path.

With GstRtspServer.RTSPSessionMedia.prototype.get_transport and GstRtspServer.RTSPSessionMedia.prototype.set_transport the transports of a GstRtspServer.RTSPStream of the managed GstRtspServer.RTSPMedia can be retrieved and configured.

Use GstRtspServer.RTSPSessionMedia.prototype.set_state to control the media state and transports.

Last reviewed on 2013-07-16 (1.0.0)
