tcpclientsink

Example launch line

 # server:
 nc -l -p 3000
 # client:
 gst-launch-1.0 fdsink fd=1 ! tcpclientsink port=3000

everything you type in the client is shown on the server (fd=1 means standard input which is the command line input file descriptor)
