Mpeg TS helper library

This library should be linked to by getting cflags and libs from gstreamer-plugins-bad-{{ gst_api_version }}.pc and adding -lgstmpegts-{{ gst_api_version }} to the library flags.

NOTE: This library API is considered unstable
