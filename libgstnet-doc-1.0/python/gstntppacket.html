Creates a new GstNtpPacket (not introspectable) from a buffer received over the network. The caller is responsible for ensuring that buffer is at least GST_NTP_PACKET_SIZE (not introspectable) bytes long.

If buffer is None, the local and remote times will be set to Gst.CLOCK_TIME_NONE.

MT safe. Caller owns return value (gst_ntp_packet_free to free).
