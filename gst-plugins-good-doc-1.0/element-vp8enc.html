vp8enc

This element encodes raw video into a VP8 stream. VP8 is a royalty-free video codec maintained by Google . It's the successor of On2 VP3, which was the base of the Theora video codec.

To control the quality of the encoding, the GstVP8Enc::target-bitrate, GstVP8Enc::min-quantizer, GstVP8Enc::max-quantizer or GstVP8Enc::cq-level properties can be used. Which one is used depends on the mode selected by the GstVP8Enc::end-usage property. See Encoder Parameters for explanation, examples for useful encoding parameters and more details on the encoding parameters.

Example pipeline

 gst-launch-1.0 -v videotestsrc num-buffers=1000 ! vp8enc ! webmmux ! filesink location=videotestsrc.webm
 ]| This example pipeline will encode a test video source to VP8 muxed in an
 WebM container.
