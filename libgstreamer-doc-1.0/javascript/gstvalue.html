GST_TYPE_LIST (not introspectable)

a GObject.Value type that represents an unordered list of GObject.Value values. This is used for example to express a list of possible values for a field in a caps structure, like a list of possible sample rates, of which only one will be chosen in the end. This means that all values in the list are meaningful on their own.

GST_TYPE_ARRAY (not introspectable)

a GObject.Value type that represents an ordered list of GObject.Value values. This is used to express a set of values that is meaningful only in their specific combination and order of values. Each value on its own is not particularly meaningful, only the ordered array in its entirety is meaningful. This is used for example to express channel layouts for multichannel audio where each channel needs to be mapped to a position in the room.

GST_TYPE_FRACTION (not introspectable)

a GObject.Value type that represents a fraction of an integer numerator over an integer denominator

GST_TYPE_BITMASK (not introspectable)

a GObject.Value type that represents a 64-bit bitmask.

GST_TYPE_FLAG_SET (not introspectable)

a GObject.Value type that represents a 32-bit flag bitfield, with 32-bit mask indicating which of the bits in the field are explicitly set. Useful for negotiation.

GST_TYPE_G_THREAD (not introspectable)

a boxed GObject.Value type for GLib.Thread that represents a thread.
