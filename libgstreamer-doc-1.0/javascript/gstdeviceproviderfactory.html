GstDeviceProviderFactory

Gst.DeviceProviderFactory is used to create instances of device providers. A GstDeviceProviderfactory can be added to a Gst.Plugin as it is also a Gst.PluginFeature.

Use the Gst.DeviceProviderFactory.prototype.find and Gst.DeviceProviderFactory.prototype.get functions to create device provider instances or use Gst.DeviceProviderFactory.prototype.get_by_name as a convenient shortcut.
