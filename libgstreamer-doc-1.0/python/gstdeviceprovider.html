GstDeviceProvider

A Gst.DeviceProvider subclass is provided by a plugin that handles devices if there is a way to programatically list connected devices. It can also optionally provide updates to the list of connected devices.

Each Gst.DeviceProvider subclass is a singleton, a plugin should normally provide a single subclass for all devices.

Applications would normally use a Gst.DeviceMonitor to monitor devices from all relevant providers.
