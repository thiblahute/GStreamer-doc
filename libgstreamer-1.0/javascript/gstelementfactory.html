GstElementFactory

Gst.ElementFactory is used to create instances of elements. A GstElementFactory can be added to a Gst.Plugin as it is also a Gst.PluginFeature.

Use the Gst.ElementFactory.prototype.find and Gst.ElementFactory.prototype.create functions to create element instances or use Gst.ElementFactory.prototype.make as a convenient shortcut.

The following code example shows you how to create a GstFileSrc element.

Using an element factory

   #include &lt;gst/gst.h&gt;

   GstElement *src;
   GstElementFactory *srcfactory;

   gst_init (&amp;argc, &amp;argv);

   srcfactory = gst_element_factory_find ("filesrc");
   g_return_if_fail (srcfactory != NULL);
   src = gst_element_factory_create (srcfactory, "src");
   g_return_if_fail (src != NULL);
   ...
