GstTagList

List of tags and values used to describe media metadata.

Strings in structures must be ASCII or UTF-8 encoded. Other encodings are not allowed. Strings must not be empty or null.

Gst.TagList.prototype.get_string 

function Gst.TagList.prototype.get_string(tag: String): {
    // javascript wrapper for 'gst_tag_list_get_string'
}

Copies the contents for the given tag into the value, possibly merging multiple values into one if multiple values are associated with the tag.

Use gst_tag_list_get_string_index (list, tag, 0, value) if you want to retrieve the first string associated with this tag unmodified.

The resulting string in value will be in UTF-8 encoding and should be freed by the caller using g_free when no longer needed. The returned string is also guaranteed to be non-%NULL and non-empty.

Free-function: g_free

Gst.prototype.tag_register 

function Gst.prototype.tag_register(name: String, flag: Gst.TagFlag, type: GObject.Type, nick: String, blurb: String, func: Gst.TagMergeFunc): {
    // javascript wrapper for 'gst_tag_register'
}

Registers a new tag type for the use with GStreamer's type system. If a type with that name is already registered, that one is used. The old registration may have used a different type however. So don't rely on your supplied values.

Important: if you do not supply a merge function the implication will be that there can only be one single value for this tag in a tag list and any additional values will silently be discarded when being added (unless Gst.TagMergeMode.prototype.replace, Gst.TagMergeMode.prototype.replace_all, or Gst.TagMergeMode.prototype.prepend is used as merge mode, in which case the new value will replace the old one in the list).

The merge function will be called from Gst.TagList.prototype.copy_value when it is required that one or more values for a tag be condensed into one single value. This may happen from Gst.TagList.prototype.get_string, Gst.TagList.prototype.get_int, Gst.TagList.prototype.get_double etc. What will happen exactly in that case depends on how the tag was registered and if a merge function was supplied and if so which one.

Two default merge functions are provided: Gst.prototype.tag_merge_use_first and Gst.prototype.tag_merge_strings_with_comma.

the type this data is in
