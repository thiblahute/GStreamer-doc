GstTracerRecord

Tracing modules will create instances of this class to announce the data they will log and create a log formatter.

gst_tracer_record_log 

#define gst_tracer_record_log(...) G_STMT_START {} G_STMT_END

Serialzes the trace event into the log.

Right now this is using the gstreamer debug log with the level TRACE (7) and the category "GST_TRACER".

Please note that this is still under discussion and subject to change.

GstTracerValueScope

Tracing record will contain fields that contain a meassured value or extra meta-data. One such meta data are values that tell where a measurement was taken. This enumerating declares to which scope such a meta data field relates to. If it is e.g. GST_TRACER_VALUE_SCOPE_PAD, then each of the log events may contain values for different GstPad.
