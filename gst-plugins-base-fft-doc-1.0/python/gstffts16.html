gst_fft_s16_new (not introspectable) 

def gst_fft_s16_new (not introspectable) (len, inverse):
    #python wrapper for 'gst_fft_s16_new'

This returns a new GstFft.FFTS16 instance with the given parameters. It makes sense to keep one instance for several calls for speed reasons.

len must be even and to get the best performance a product of 2, 3 and 5. To get the next number with this characteristics use GstFft.fft_next_fast_length.

def GstFft.FFTS16.fft (self, timedata, freqdata):
    #python wrapper for 'gst_fft_s16_fft'

This performs the FFT on timedata and puts the result in freqdata.

timedata must have as many samples as specified with the len parameter while allocating the GstFft.FFTS16 instance with gst_fft_s16_new (not introspectable).

freqdata must be large enough to hold len/2 + 1 GstFft.FFTS16Complex frequency domain samples.
