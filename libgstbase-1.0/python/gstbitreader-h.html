GstBitReader

GstBase.BitReader provides a bit reader that can read any number of bits from a memory buffer. It provides functions for reading any number of bits into 8, 16, 32 and 64 bit variables.

Function Macros

GstBase.BIT_READER 

#define GST_BIT_READER(reader) ((GstBitReader *) (reader))

GstBase.BIT_READER_INIT 

#define GST_BIT_READER_INIT(data, size) {data, size, 0, 0}

A GstBase.BitReader must be initialized with this macro, before it can be used. This macro can used be to initialize a variable, but it cannot be assigned to a variable. In that case you have to use GstBase.BitReader.init.
