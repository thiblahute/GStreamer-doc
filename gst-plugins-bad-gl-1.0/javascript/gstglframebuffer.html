GstGLFramebuffer

A GstGL.GLFramebuffer represents and holds an OpenGL framebuffer object with it's associated attachments.

A GstGL.GLFramebuffer can be created with GstGL.GLFramebuffer.prototype.new or GstGL.GLFramebuffer.prototype.new_with_default_depth and bound with GstGL.GLFramebuffer.prototype.bind. Other resources can be bound with GstGL.GLFramebuffer.prototype.attach

Note: OpenGL framebuffers are not shareable resources so cannot be used between multiple OpenGL contexts.
