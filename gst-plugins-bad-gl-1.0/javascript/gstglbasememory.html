GstGlBaseMemory

GstGLBaseMemory is a Gst.Memory subclass providing the basis of support for the mapping of GL buffers.

Data is uploaded or downloaded from the GPU as is necessary.

Constants

GstGL.MAP_GL

Flag indicating that we should map the GL object instead of to system memory.

Combining GstGL.MAP_GL with Gst.MapFlags.prototype.write has the same semantics as though you are writing to OpenGL. Conversely, combining GstGL.MAP_GL with Gst.MapFlags.prototype.read has the same semantics as though you are reading from OpenGL.
