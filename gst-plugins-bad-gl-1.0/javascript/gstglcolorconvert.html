GstGlColorConvert

GstGL.GLColorConvert is an object that converts between color spaces and/or formats using OpenGL Shaders.

A GstGL.GLColorConvert can be created with GstGL.GLColorConvert.prototype.new, the configuration negotiated with GstGL.GLColorConvert.prototype.transform_caps and the conversion performed with GstGL.GLColorConvert.prototype.perform.

The glcolorconvertelement provides a GStreamer element that uses GstGL.GLColorConvert to convert between video formats and color spaces.
